**nekotk** – a friendly, lightweight toolkit for quick demos and examples.  
It ships with a tiny collection of utilities that make prototyping in Python a breeze.

## ✨ Features
* Simple, well‑documented API
* Zero‑dependency core (optional extras available)
* Ready‑to‑run examples in the `examples/` folder

## 📦 Installation
```bash
pip install nekotk
```

## 🛠️ Usage
```python
from nekotk import demo

demo.run()
```
